## About EB-1A Petition Builder

EB-1A Petition Builder is a browser-based environment for putting together EB-1A immigration petition materials. It is built to help users arrange supporting evidence, develop case narratives, and produce polished petition documents for USCIS-focused workflows.

The application combines AI-guided organization with document generation, making it easier to evaluate evidence, compare materials, and assemble submission-ready outputs. With LaTeX and PDF export support, it fits applicants, advisors, and teams that want a structured way to turn source materials into a complete petition package.

## Features

- AI-assisted evidence grouping for organizing supporting files
- Petition narrative drafting tools for composing case sections
- Strength scoring and gap review to assess overall coverage
- LaTeX and PDF export generation for document assembly
- Support for multilingual evidence in mixed-language source materials
- Collaboration tools for shared review and editing
- Version-control-friendly workflow for tracking edits over time
- Web-based interface for browser-first use on modern platforms

## Usage

A typical workflow looks like this:

1. Add EB-1A evidence items and related reference files.
2. Group documents by category and review the AI-assisted strength analysis.
3. Draft or refine the petition narrative using the built-in writing tools.
4. Generate LaTeX output when you need a structured manuscript.
5. Export the final package as PDF for review or submission prep.
6. Use collaboration and version tracking when multiple contributors are involved.

For best results, keep your source documents well labeled and update the case structure as new evidence is added.
